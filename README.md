# Keep Moving - VSCode Extension

## Description
Keep Moving is a VSCode extension designed to remind users to stand up and move periodically. The extension displays a visual notification every 10 minutes to encourage healthier habits while working.

## Key Features
- 🕒 Automatic reminders every 10 minutes
- 💪 Engaging visual interface with animations
- ⏰ Snooze option to delay reminders
- 🔄 Ability to enable/disable reminders
- 📊 Informative status bar indicator
